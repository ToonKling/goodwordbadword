# Good word - Bad Word

## DB

PSQL
```
create table dictionary (
    id int PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY, 
    word text NOT NULL, 
    definition text NOT NULL
);
create table rating (
    id            int     PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
    dictionary_id int     NOT NULL REFERENCES dictionary,
    rating        int     NOT NULL
);
```
BASH
```
curl "https://raw.githubusercontent.com/matthewreagan/WebstersEnglishDictionary/refs/heads/master/dictionary.json" > dictionary.json
jq -r 'to_entries[] | [.key, .value] | @tsv' dictionary.json  > dictionary.tsv
psql "postgresql://user:password@localhost:5432/mydb" -c "\copy dictionary(word, definition) FROM 'dictionary.tsv' WITH (FORMAT csv, DELIMITER E'\t')"
```
